AI를 활용한 보안,
AI를 위한 보안.

PRαXIS Research는 AI가 만드는 새로운 시스템, 데이터, 에이전트 환경을 이해하고 그 환경에 필요한 보안 기술을 연구·구현합니다.

01

AI를 활용한 보안

신경망 기반 암호 생성·검증과 자동 안전성 평가

02

AI를 위한 보안

프롬프트, 에이전트, 학습·추론 데이터 흐름 보호

Belief

AI 보안은 특정 제품 하나가 아니라 변화하는 환경을 다루는 기술입니다.

생성형 AI, 에이전트, 분산 학습, 자동화된 분석 환경은 보안의 대상과 방법을 동시에 바꾸고 있습니다. PRαXIS Research는 이 변화에 맞춰 연구, 구현, 검증이 함께 움직이는 보안 기술을 만듭니다.

AI로 보안을 고도화합니다.

신경망 기반 생성, 자동 검증, 민감정보 탐지처럼 AI를 활용해 보안 기술의 탐색과 적용 속도를 높입니다.

AI 시스템 자체를 안전하게 만듭니다.

프롬프트, 에이전트, 학습·추론 데이터, 외부 모델 연동 과정에서 생기는 위험을 줄이는 AI를 위한 보안 기술을 설계합니다.

Scope

AI 보안의 여러 문제를 연구하고 제품화합니다.

Technology 보기
01

암호 생성·검증

해시 함수와 대칭키 암호 후보를 생성하고 안전성을 자동 검증하는 기술을 연구합니다.

02

프롬프트·에이전트 보안

AI 사용 과정에서 민감 정보와 내부 정책이 외부 모델로 노출되지 않도록 설계합니다.

03

분산 AI 데이터 보호

원본 데이터 이동을 줄이고 태스크에 필요한 정보만 안전하게 활용하는 구조를 만듭니다.

04

AI 보안 평가와 확장

AI 시스템 평가, 보안 검증, 레드팀 자동화까지 확장 가능한 연구 방향을 준비합니다.

Praxis

논문과 아이디어를 실제 환경에서 작동하는 기술로 옮깁니다.

  1. Research

    문제를 정의합니다.

    AI가 바꾸는 보안 환경에서 새로 생기는 위험과 기회를 연구합니다.

  2. Prototype

    작동하는 형태로 만듭니다.

    아이디어를 실험 가능한 모듈, API, 워크플로우로 구현합니다.

  3. Validation

    검증하고 개선합니다.

    자동 검증과 실험 평가를 통해 성능과 안정성을 함께 확인합니다.

  4. Application

    현장 적용을 논의합니다.

    PoC, 공동 연구, 도입 검토를 통해 실제 환경에 맞는 형태로 확장합니다.

Direction

PRαXIS Research는 AI 보안 회사입니다.

우리는 AI를 활용해 보안 기술을 만들고, AI 시스템이 안전하게 운영될 수 있도록 필요한 기술을 확장해갑니다.
